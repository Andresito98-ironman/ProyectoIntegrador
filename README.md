# Proyecto Integrador en Python

Este proyecto forma parte de un trabajo académico y tiene como objetivo implementar un sistema modular en Python utilizando buenas prácticas de programación.

## 📂 Estructura del proyecto
- `main.py`: Archivo principal que ejecuta el programa.
- `funciones.py`: Contiene las funciones auxiliares y la lógica modular.
- `__pycache__/`: Carpeta generada automáticamente por Python para almacenar archivos compilados.
